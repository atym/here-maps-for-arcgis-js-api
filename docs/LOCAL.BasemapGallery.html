LOCAL. BasemapGallery

This class allows for using Non-ArcGIS tiled services in the BasemapGallery widjet

new BasemapGallery(options, options.map, options.services, options.titlePaneID, options.basemapGalleryID, options.defaultServiceURL, options.showArcGISBasemaps, options.bingMapsKeyopt)
